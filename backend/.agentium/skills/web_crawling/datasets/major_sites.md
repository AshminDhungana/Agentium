# Major Websites by Category (~100)

A curated starting-point list for agents picking good crawling/search sources by
topic. Use these as `url` seeds for the `web_crawler` tool (or `web_fetch` for a
single page). Verify a site is current and appropriate before crawling; always
respect its `robots.txt`.

## General Reference & Search
- **Wikipedia** — wikipedia.org — open encyclopedia, broad factual coverage.
- **Wikidata** — wikidata.org — structured linked open data.
- **Internet Archive** — archive.org — Wayback Machine, public-domain media.
- **Google** — google.com — web search (results pages are bot-restricted).
- **Bing** — bing.com — search + index.
- **DuckDuckGo** — duckduckgo.com — privacy-focused search.
- **Stack Exchange** — stackexchange.com — network of Q&A sites.
